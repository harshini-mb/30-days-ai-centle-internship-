# Day 2 – Hello, Cortex!

## Overview

For Day 2 of the 30 Days of AI Challenge, I built a Streamlit application that connects to Snowflake Cortex
and generates AI-powered responses from user prompts.

The application allows users to enter a question or prompt, sends it to a Large Language Model (LLM) 
using Snowflake Cortex's `AI_COMPLETE` function, and displays the generated response in the Streamlit interface.

---

## Technologies Used

* Python
* Streamlit
* Snowflake
* Snowpark
* Snowflake Cortex AI
* JSON

---

## What I Learned

* How to use Snowflake Cortex AI models
* Running LLM inference directly inside Snowflake
* Using the `AI_COMPLETE()` function
* Creating interactive user input forms with Streamlit
* Parsing JSON responses from AI models
* Displaying AI-generated content in a web application

---

## Features

* User prompt input field
* AI response generation
* Snowflake Cortex integration
* Streamlit-based user interface
* JSON response handling

---

## Challenges Faced

* Understanding how Snowflake Cortex processes prompts
* Learning how AI responses are returned as JSON
* Parsing and displaying the generated output correctly

---

## Outcome

Successfully built a Streamlit application that accepts user prompts and generates responses using Snowflake Cortex AI models.
