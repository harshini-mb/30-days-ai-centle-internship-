# 📅 Day 10 – Your First Chatbot (with State)

## 🎯 Objective
Build a chatbot that remembers the conversation using Streamlit Session State and Chat Elements.

## 🛠️ What I Learned
- Using `st.chat_input()` for user messages
- Using `st.chat_message()` for chat bubbles
- Understanding Session State
- Storing conversation history
- Building a chatbot interface
- Connecting an AI model to a Streamlit application

## 🚀 Features
- User-friendly chat interface
- Persistent conversation history
- AI-generated responses
- Session State memory management
- Real-time message display

## 📚 Key Concepts
- `st.session_state`
- `st.chat_input()`
- `st.chat_message()`
- Message storage using lists and dictionaries
- AI response handling

## 💡 How It Works
1. Initialize an empty message list in Session State.
2. Display previous messages from history.
3. Accept new user input through chat input.
4. Generate an AI response.
5. Save both user and assistant messages.
6. Redisplay the full conversation on every rerun.

## 🏆 Outcome
Successfully created my first chatbot that can remember previous messages and maintain conversation history throughout the session.
