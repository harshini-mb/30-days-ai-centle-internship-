# Day 7 – Theming and Layout

## Overview

On Day 7 of the 30 Days of AI Challenge, I enhanced my LinkedIn Post Generator by focusing on user experience, application design, and interface organization.

The goal was to transform a functional AI application into a polished and professional-looking web app using Streamlit's theming and layout features.

## What I Learned

* Customizing Streamlit application themes
* Creating a dark-mode user interface
* Organizing controls using the sidebar
* Improving visual hierarchy with containers
* Displaying progress using status indicators
* Building a cleaner and more professional user experience

## Features Implemented

### Dark Theme Configuration

Configured a custom Streamlit theme using `config.toml` with:

* Dark background
* Custom accent colors
* Styled buttons
* Improved typography

### Sidebar Navigation

Moved application settings to a dedicated sidebar:

* Tone selection
* Word count selection
* App information

### Status UI

Implemented `st.status()` to display progress updates during AI generation:

1. Analyzing prompt requirements
2. Contacting AI model
3. Generating response
4. Completion status

### Output Container

Displayed generated LinkedIn posts inside a bordered container for better readability and visual separation.

## Technologies Used

* Python
* Streamlit
* Snowflake Cortex AI
* Snowpark

## Key Takeaway

A great AI application is not only about powerful models but also about providing a clean, intuitive, and engaging user experience. Small design improvements can significantly enhance usability and professionalism.

## Outcome

Successfully built a themed and structured LinkedIn Post Generator with improved layout, status tracking, and user-friendly design.
